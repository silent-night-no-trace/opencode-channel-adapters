# Changelog

All notable changes to this project will be documented in this file.

This project follows semantic versioning for published packages.

## 0.2.0

- Added optional HTTP(S) proxy support for Telegram Bot API and Discord REST traffic.
- Added configurable Telegram polling retry delays for transient network failures.
- Added Telegram session management commands and inline session selection helpers.
- Expanded configuration, example config, and local debug documentation for the new runtime options.

## 0.1.0

- Initial channel adapter workspace with shared core runtime.
- Added Telegram, Discord, and Feishu/Lark adapter packages.
- Added unified JSON/JSONC configuration loading and environment overrides.
- Added local development, debug, and contribution documentation.
